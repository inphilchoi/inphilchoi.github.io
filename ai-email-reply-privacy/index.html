AI 메일답장은 개발자 운영 서버가 없습니다. 계정 정보·메일 내용·사용 기록을 개발자가 수집, 저장, 열람하는 일은 없으며 모든 처리는 사용자의 기기에서 직접 이루어집니다.

1.수집하는 개인정보

없습니다. 회원가입이 없고, 분석 도구(애널리틱스)·광고 SDK·추적 기술을 사용하지 않습니다.

2.메일 데이터의 처리

  • 메일함 연동(IMAP/SMTP 또는 Gmail API)은 사용자의 기기에서 메일 서버로 직접 통신합니다. 중간 서버를 거치지 않습니다.
  • 메일 내용은 화면 표시·요약·초안 생성 목적으로 기기 메모리에서만 처리되며, 기기 밖으로 전송되지 않습니다(아래 3항의 선택 기능 제외).
  • 메일 서버 접속 정보(이메일 주소, 앱 비밀번호 등)는 기기에만 저장됩니다.

3.AI 처리 (선택 기능)

기본 상태에서 요약·초안 생성은 인터넷 없이 기기 안의 규칙 기반 엔진으로 동작합니다.

사용자가 직접 본인의 API 키를 설정한 경우에 한해, 요약·초안 생성 시 해당 메일 본문이 사용자가 선택한 AI 제공사(Google Gemini 또는 Groq) API로 전송됩니다. 이는 전적으로 옵트인이며, 키를 삭제하면 즉시 중단됩니다. 전송된 데이터의 처리는 각 제공사의 정책을 따릅니다: Google Gemini API 약관 · Groq 개인정보처리방침

  • "내 문체 학습"은 사용자가 명시적으로 동의한 경우에만 동작합니다.
  • 보낸 메일에서 인사말·맺음말·문장 스타일 요약 정보만 기기에 저장하며, 메일 원문은 저장하지 않습니다.
  • 동의를 철회하면 학습된 정보가 즉시 삭제됩니다.

Privacy Policy (English Summary)

AI Email Reply ("the app") operates no developer server and collects no personal data. No sign-up, no analytics, no ads, no trackers.

  • Mail data: the app connects directly from your device to your mail server (IMAP/SMTP or Gmail API). Email content is processed on-device only.
  • AI processing (optional): only if you enter your own API key, email bodies are sent to the AI provider you chose (Google Gemini or Groq) to generate summaries/drafts, governed by their policies.
  • Writing-style learning (opt-in): only style summaries (greeting/closing/tone) are stored on-device; withdrawing consent deletes them.
  • Google user data: use of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.
  • Deletion: all locally stored data is removed when you delete the app.
